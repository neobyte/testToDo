# ToDo app

## Install and setup


Requirements
- [Latest Node](https://nodejs.org/en/).
- [Latest Laravel Mix (6) and Vue 3](https://github.com/JeffreyWay/laravel-mix/blob/master/UPGRADE.md).
```angular2html
npm i -D laravel-mix@latest vue@next @vue/compiler-sfc vue-loader@next
```
- [Latest Tailwind](https://laravel.com/docs/8.x/mix#tailwindcss).
- PHP > 7.3.0

## Setup app

- Run ```composer install```
- Run migrations ```php artisan migrate```
